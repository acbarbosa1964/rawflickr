type TComments = class(TRESTApi)

Description

Implements flickr.photos.comments.* and flickr.photosets.comments.*

TComments is polimorphic and depends of BaseGroup and BaseIdName to know which group and Id names to use on calls to the Flickr API

property BaseGroup: String read FBaseGroup;

Name of the base group of calls. BaseGroup is used to build the 'method' parameter in the call to the web service; p.e. for flickr.photosets.comments.* you would make:

BaseGroup := 'photosets';

Both TPhotos and TPhotosets do this themselves when they create their Comments property; you don't need to do it yourself.

property BaseIdName: String read FBaseIdName;

Name of the base ID to use in the call to the web service. BaseIdName is used to build the 'xxx_id' parameter for the web service call; p.e. flickr.photos.comments.* uses a parameter called photo_id, so you would do:

Comments.BaseIdName := 'photo';.

Note: Both TPhotos and TPhotosets do this themselves when they create their Comments property; you don't need to do it yourself.
